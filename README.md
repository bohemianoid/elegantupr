# elegantupr
Elegant Visualisation of ecoinvent Unit Processes

## Running Locally
```
$ git clone https://github.com/simonroth/elegantupr.git
$ cd elegantupr
$ npm install
$ npm run build
$ npm start
```

elegantupr should now be running on [localhost:5001](http://localhost:5001).

## Development
browserify
```
npm run build
```

watchify
```
npm run watch
```

## Deploying to Heroku
```
$ heroku create
$ git push heroku master

$ heroku open
```
